## 🛠 Troubleshooting Tips

If you have trouble, try these common fixes:

- Make sure your microphone is plugged in and working.
- Check Windows microphone permissions:
  - Go to Settings > Privacy > Microphone.
  - Ensure the app is allowed to access the microphone.
- Restart the computer if the app doesn’t start.
- Confirm you downloaded the right installer for Windows.
- Close other apps that might use the microphone.
- Try speaking more slowly and clearly.
- Update your audio drivers via the Device Manager.
